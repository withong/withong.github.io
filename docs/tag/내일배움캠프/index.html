JWT란?

1. JWT 정의 JWT는 JSON Web Token의 약자로, JSON 형식의 데이터를 안전하게 전송하기 위한 표준이다. 이 표준은 인터넷 상에서의 인증 및 권한 부여와 관련된 시스템에서 주로 사용된다. 예를 들어, 사용자가 서버에 로그인하면 서버는 JWT를 생성하여 사용자에게 제공하고, 사용자는 이 JWT를 통해 인증을 유지할 수 있다.
